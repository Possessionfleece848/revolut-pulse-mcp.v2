## 🧭 What this app does

revolut-pulse-mcp.v2 gives Claude, Cursor, and other AI tools live stock and crypto prices.

It uses Yahoo Finance for stocks and Binance for crypto. It also checks Revolut market access, so you can see if a symbol is likely available there.

This app is for Windows users who want a simple local setup with no API key.
